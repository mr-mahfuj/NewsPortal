# How to Run NewsPortal


### Backend
```bash
cd backend
./start.sh
```
Backend runs at: http://localhost:8000  
API Docs: http://localhost:8000/docs

### Frontend
```bash
cd news-portal
npm install
npm run dev
```
Frontend runs at: http://localhost:5173
